Program

Program selects stages, changes their behavior, stores presets, receives four external sources, and collects every head's CV into one polyphonic output.

Selection and bulk editing

ControlBehavior
SelectMove the edit selection left or right. Hold to autoscroll; selection wraps across the reachable stage chain.
Stage displayShows the reachable stage count. An exclamation mark indicates a truncated or invalid chain. AL means the next gesture will edit all stages.
ClearReset all stage programming to defaults without moving voltage or time sliders.

Hold Select while applying a modifier to edit every stage. For mouse use, choose Apply next edit to ALL stages from the context menu.

The five Limited buttons select a 2 V slider span with octave offset −2 through +2. Limited is independent of the Full/Half range selection.

Operating mode

ModifierEffect when set
StopThe head completes this interval and stops. A new Start continues at the following stage.
SustainThe head holds this stage while its Start input remains high.
EnableThe head waits on entry until its Start input exceeds 5 V.
First / LastMark region boundaries used by Head's First–Last loop mode.

Interval time and pulses

Choose one of four ranges per stage: 0.002–0.03 s, 0.02–0.3 s, 0.2–3 s, or 2–30 s. Time Source selects the Stage4 time slider or one of external inputs A–D; in external mode the time slider chooses the input quartile.

Pulse 1 and Pulse 2 flag the selected stage for the corresponding Head outputs. Pulse Retrig defaults on, inserting a brief low notch between adjacent flagged stages. Turn it off to merge adjacent stages into one gate.

Presets, key, and scale

For presets, press Load or Save and then 1–12. Presets store stage programming and slider values in the patch. Recalled slider values use takeover: a physical slider does not regain control until it crosses the stored value.

Press Key and then 1–12 for C through B. Press Scale and then 10, 11, or 12 for Major, Minor, or Chromatic.

Context menu

  • Arm one-shot all-stage editing.
  • Choose slew level 1 from 10%, 25%, or 40% of the interval.
  • Choose slew level 2 from 50%, 75%, or 100% of the interval.
  • Choose self-normalizing 0–10 V address scaling or fixed 0.5 V per stage.
  • Slope law is currently Linear; Exponential is reserved.
