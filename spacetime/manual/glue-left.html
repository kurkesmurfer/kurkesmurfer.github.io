Glue Left

Glue Left begins the right-hand fragment of a separated SpaceTime chain. Pair it with Glue Right on the same link number.

Placement

Place Glue Left immediately to the left of the remote fragment. It can continue either a head-side or stage-side cut:

Glue Left→MIDI / Program / Stage4

Link and readout

Choose Link 1–8 in the context menu. The readout combines the link number with H for a head-side fragment, S for a stage-side fragment, or a dash when no side is detected.

LightMeaning
GreenA unique, compatible partner is linked.
RedWaiting, mismatched, reversed, or duplicated link.
OffNo valid SpaceTime neighbor.

Scope

The pair transports only SpaceTime's internal expander messages inside one VCV Rack patch. It is not a cableless audio/CV connection and is not compatible with third-party expanders.

The context menu also reports detected side, link status, and queue drops for diagnosis.
